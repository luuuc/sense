# Bench2 Autonomous Improvement Loop - Simplified Approach

This directory contains a simplified approach to running the bench2 autonomous improvement loop using prompt files that any tool can read and follow.

## Files

- `prompt-loop-1.md` - Analysis phase instructions
- `prompt-loop-2.md` - Improvement phase instructions  
- `prompt-loop-3.md` - Validation phase instructions
- `HOW_TO_USE.md` - Guide for using these prompts

## Approach

Instead of complex skill implementations, simply:

1. Read the appropriate prompt file (`prompt-loop-1.md`, `prompt-loop-2.md`, or `prompt-loop-3.md`)
2. Follow the instructions contained within
3. Execute the suggested commands
4. Move to the next prompt when the current phase is complete

## Benefits

- Works with any AI tool that can read and follow instructions
- No complex setup or configuration required
- Easy to understand and modify
- Maintains all the benefits of the original improvement plan
- Reduces complexity while preserving effectiveness

## Usage

To start the improvement process:

```bash
# Read and follow the first loop instructions
cat prompt-loop-1.md
# Then follow the steps outlined in the file
```

See `HOW_TO_USE.md` for complete usage instructions.
