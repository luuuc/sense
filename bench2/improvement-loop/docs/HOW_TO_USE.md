# How to Use the Bench2 Improvement Prompts

This document explains how to use the simplified prompt-based approach for running the bench2 autonomous improvement loop.

## Overview

Instead of complex skill implementations, we've created simple prompt files that any AI assistant can read and follow to execute the improvement process:

- `prompt-loop-1.md` - Analysis phase: Extract quality markers and reduce false positives
- `prompt-loop-2.md` - Improvement phase: Generate semantic enhancements and better checks
- `prompt-loop-3.md` - Validation phase: Optimize scoring weights for accuracy

## Getting Started

1. Simply read the appropriate prompt file for the phase you want to execute
2. Follow the instructions in the prompt
3. Execute the commands as directed
4. Move to the next prompt after completing each phase

## Usage Example

```bash
# To start with Loop 1 (Analysis Phase)
cat prompt-loop-1.md

# Follow the instructions in the file
# Run the suggested commands
# Proceed to next loop when ready
```

## Prerequisites

Before starting, ensure you have:

1. bench2 environment properly set up
2. All dependencies installed
3. Access to required models (Sonnet for runs, Opus for analysis)
4. Basic familiarity with the bench2 system

## Process Flow

1. **Loop 1 (prompt-loop-1.md)**: 3 iterations focusing on verifiability
   - Goal: Replace keyword checks with verifiable checks
   - Outcome: 15% reduction in false positives per iteration

2. **Loop 2 (prompt-loop-2.md)**: 3 iterations focusing on semantic depth
   - Goal: Generate semantic checks and improve tool usage
   - Outcome: 25% increase in MCP tool usage

3. **Loop 3 (prompt-loop-3.md)**: 3 iterations focusing on weight optimization
   - Goal: Auto-tune scoring weights for maximum accuracy
   - Outcome: 0.2+ point separation between tools

## Completion Criteria

The full improvement process is complete when:

- All 9 iterations across 3 loops are finished
- Final Sense scores are in 0.8-0.9 range
- Final baseline scores are in 0.6-0.7 range
- Tool differentiation shows 0.2+ point gap
- Total execution time is ~4.5 hours
- Total cost is ~$103

## Troubleshooting

If you encounter issues:

1. Check that all prerequisites are met
2. Verify API keys and model access
3. Ensure sufficient disk space for results
4. Monitor costs to stay within budget
5. Review logs in the results/ directory
